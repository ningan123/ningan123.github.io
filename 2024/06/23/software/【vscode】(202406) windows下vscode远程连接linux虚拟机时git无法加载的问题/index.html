【vscode】(202406) windows下vscode远程连接linux虚拟机时git无法加载的问题

问题1

问题介绍

windwos下的vscode自动升级到1.89.1，linux虚拟机中的部分文件夹就无法加载git，部分文件夹可以加载，哪些文件夹可以加载、哪些文件夹不能加载是随机的

解决办法

linux虚拟机的根目录下有个文件夹，叫.vscode-server，删除该文件夹，重启vscode，该文件夹自动生成，然后就好了

问题2

问题介绍

error while fetching extensions.xhr failed

排查

open the network tab (F1 > Toggle Developer Tools) and check the network error.

发现是代理问题

打开远程linux系统下的一个目录，查看Proxy，没有东西

User下面也没有东西

打开一个windows空窗口，查找Proxy，确实有一个代理

解决办法

删除此代理，返回打开远程linux代码的窗口，在扩展里面搜索插件，例如：Go，就可以加载出来了
我都没有重启vscode，就正常了~

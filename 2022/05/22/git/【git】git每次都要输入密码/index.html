【git】git每次都要输入密码

windows和linux下避免git clone和push时每次都需要输入用户名和密码

git 可以用 https 方式访问也可以用 ssh 方式访问

1）https：就是你每次要输入密码那种

2）ssh：用公钥私钥 本地用ssh-keygen生成密钥，分为 私钥和公钥，私钥本地保存，公钥放到服务端
